# Overview
This library is aimed to connect C++ application to SageMath plotting system. The library will generate Sage scripts, then execute it, and save plot to specified destination.
# Functions
```cpp
void plot(field& f, settings::files info)
void point(Karabinerhaken<coord<D>>& pts, settings::files info)
void line(Karabinerhaken<coord<D>>& pts, settings::files info)
```
