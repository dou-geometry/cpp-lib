# Overview

This project is mainly to maintain a general-physics-oriented C++ library. Following is a list of our goals:

- [x] Coordination/Vectors Carrier
- [x] Linear Algebra
- [x] Plotting (though SageMath)
- [ ] Dynamic Systems
  - [ ] RK4
- [ ] Numerical Integration
- [ ] boxCounting
